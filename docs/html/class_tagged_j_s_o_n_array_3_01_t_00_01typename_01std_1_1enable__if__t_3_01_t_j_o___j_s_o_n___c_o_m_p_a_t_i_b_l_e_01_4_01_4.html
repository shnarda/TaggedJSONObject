Detailed Description

template<typename T>
class TaggedJSONArray< T, typename std::enable_if_t< TJO_JSON_COMPATIBLE > >

The TaggedJSONArray class variation that accepts JSON compatible data as its container.

This class is used for storing JSON arrays with a preknown type (QVariantArrays can be used for mixed types)
Main usage for this class is to treat JSON data as class member for the tagged objects that has been generated from the TJO_DEFINE_JSON_TAGGED_OBJECT() macro.
TaggedJSONArray can store all types of data JSON format accepts but TaggedJSONArray<T, typename std::enable_if_t< std::is_constructible_v<T, QJsonValue, bool>&& std::is_constructible_v<T, QString, bool>&& std::is_constructible_v<T, QByteArray, bool>&& std::is_constructible_v <T, QJsonObject, bool>>> is more suitable for chaining JSON objects within JSON arrays.
